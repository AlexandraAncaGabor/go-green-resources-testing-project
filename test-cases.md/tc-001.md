# **[TC-001] - Verify logo redirects to main page**

### **Description**

Clicking the logo redirects the user to the main page.

### **Types of testing:**

- Functional testing,
- Positive testing.

| #   | **Test steps**    | **Expected result**                     |
| --- | ----------------- | --------------------------------------- |
| 1   | Open the web page | The page opens                          |
| 2   | Click on the logo | The user is redirected to the main page |

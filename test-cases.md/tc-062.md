# **[TC-062] - Verify text alignment on ‘Iaz de decantare Bozânta’ page**

### **Description**

Opening the page "Iaz de decantare Bozânta" will show an alligned text.

### **Types of testing:**

- Functional testing;
- Positive testing.

| #   | **Test steps**                         | **Expected result**                     |
| --- | -------------------------------------- | --------------------------------------- |
| 1   | Open the web page                      | The page opens                          |
| 2   | Go to the menu and select "Activități" | "Activități"  page opens                |
| 3   | Click on "Iaz de decantare Bozânta"    | The page opens and is properly alligned |
